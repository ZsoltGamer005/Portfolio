Szia, Zsolt vagyok. Digitális élményeket építek. Frontend fejlesztőként a küldetésem, hogy a letisztult designt pixelpontos, gyors és jól működő kóddá alakítsam. Imádom az olyan weboldalakat és applikációkat, amiket nemcsak jó nézni, de élmény használni is.

Ki vagyok én?

Mindig is érdekelt a technológia és a vizualitás találkozása, így a frontend fejlesztésben találtam meg önmagam. Az elmúlt 3 évben azon dolgoztam, hogy a designerek merész látomásait életre keltsem a böngészőben.

Nemcsak kódot írok: figyelek a felhasználói élményre (UX), a reszponzivitásra (hogy mobilon is tökéletes legyen) és a weboldalak sebességére. Ha épp nem a monitort bújom, valószínűleg egy új tech-trendet tanulmányozok, podcastot hallgatok, vagy a tökéletes eszpresszót próbálom lefőzni.

Amikkel szívesen dolgozom

  • Core: HTML5, CSS3 / SASS, JavaScript
  • Keretrendszerek: React, Next.js
  • Formázás / UI: Tailwind CSS, Material UI
  • Eszközök: Git, GitHub, Figma, VS Code

Projektek

EcoShop – E-commerce Webshop

Egy zöld, fenntartható termékeket árusító webáruház teljes frontend fejlesztése.

Egy letisztult, Kanban-tábla alapú feladatkezelő alkalmazás.
